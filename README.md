
# 🔍 Image Search Engine

A modern and responsive Image Search Engine built using **HTML, CSS, and JavaScript**. This project allows users to search for high-quality images in real time using an external image API. It features a clean UI, fast search experience, and a "Load More" functionality for browsing additional results.

## 🚀 Features

- 🔎 Search images instantly by keyword
- 🖼️ Display high-quality image results
- 📄 Load more images without refreshing the page
- ⚡ Fast API integration
- 📱 Fully responsive design
- 🎨 Clean and modern user interface
- 🔗 Click images to view them in full resolution


## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Image Search API (Unsplash)

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/image-search-engine.git
```

2. Open the project folder

```bash
cd image-search-engine
```

3. Open `index.html` in your browser.

---

## 🔑 API Configuration

If you're using an image API, add your API key inside `script.js`.

Example:

```javascript
const accessKey = "YOUR_API_KEY";
```

You can get a free API key from:

- Unsplash Developers
---

## 💻 How It Works

1. Enter a keyword in the search box.
2. Click the **Search** button.
3. Images related to the keyword will be displayed.
4. Click **Load More** to fetch additional images.
5. Click an image to view it in full size.
